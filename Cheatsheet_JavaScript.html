JavaScript Programming Cheatsheet

Class / Method Description Example
appendChild() Adds a node to the end of a parent element. var p = document.createElement("p"); p.appendChild(document.createTextNode("Hello World")); document.body.appendChild(p);
Arrays Stores multiple values in a single variable. const Beatles = ["Ringo", "Paul", "George", "John"]; // Beatles[0] = "Ringo"
Date() Creates date objects. var d = new Date("2021-01-17 13:15:30"); var d2 = new Date(2021, 0, 17);
getElementById() Selects element by ID. document.getElementById("div1").innerHTML = "Hello World";
getElementsByTagName() Returns elements by tag name. var pTags = document.getElementsByTagName("p");
document.write() Writes directly to page (not recommended). document.write("Hello World");
innerHTML Gets or sets HTML content. document.getElementById("div1").innerHTML = "<p>Hello</p>";
setAttribute() Sets attribute value. document.getElementById("img").setAttribute("src", "image.png");
removeAttribute() Removes attribute. element.removeAttribute("style");
style Changes inline CSS. element.style.color = "red";
Error Objects Handles errors using try/catch. try { throw new Error("Invalid"); } catch (err) { console.log(err.message); }
History Object Controls browser history. history.go(-1);
Location Object Info about current URL. console.log(location.hostname);
Navigator Object Browser information. console.log(navigator.appName);
Window Object Main browser object. window.open("https://example.com");
scrollTo() Scrolls window. window.scrollTo(0, 200);
